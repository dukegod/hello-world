### Array

##### Array.prototype.includes
用于查找数组中是否包含了某个项（与 indexOf 不同的是，它可以包含 NaN）。

### Math

### `...` 用法

+ 数组拼接 : [...arr1, ...arr2]
+ 类数组转为数组方式： arguments
+ 对象浅复制 类似： `object.assign()`
+ 循环 Map，Set 数据结构
